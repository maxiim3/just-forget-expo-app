# Day 0 — Project Kickoff

## Summary

**Forget** — Universal mental inbox app initialized.

## Decisions

1. **Web-first development** — No Android Studio or iOS Simulator required. Use `bunx expo start --web` for all development.

2. **Bun as package manager** — Faster than npm/yarn. Use `bun add` for regular deps, `bunx expo install` for Expo-compatible packages.

3. **Tech stack finalized:**
   - Expo SDK 52+ with Expo Router
   - NativeWind for styling (Tailwind)
   - Zustand for state
   - Supabase for backend
   - rn-swiper-list for card swiping

4. **Phase 1 focus** — MVP with capture mode only. Retrieve mode (AI chat) deferred to Phase 2.

5. **Design direction** — Hand-drawn/sketch aesthetic with Caveat or Permanent Marker fonts, large rounded borders.

## Context Established

- Core flow: `Drop → Forget → Find → Send`
- Gesture-based UI: swipe right (next), left (archive), up/down (edit)
- Card stack (Tinder-style) as primary interface
- Supabase schema: entries, labels, entry_labels
