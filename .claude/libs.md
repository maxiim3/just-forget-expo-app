# Libraries

## Core Expo

| Package | Purpose |
|---------|---------|
| expo | Expo SDK |
| expo-router | File-based routing |
| react-native-web | Web support |
| @expo/metro-runtime | Metro bundler for web |

## UI & Styling

| Package | Purpose |
|---------|---------|
| nativewind | Tailwind for React Native |
| tailwindcss | Utility-first CSS |

## Animation & Gestures

| Package | Purpose |
|---------|---------|
| react-native-reanimated | Declarative animations |
| react-native-gesture-handler | Native gesture system |
| rn-swiper-list | Tinder-style swipe cards |

## State Management

| Package | Purpose |
|---------|---------|
| zustand | Lightweight state management |

## Backend

| Package | Purpose |
|---------|---------|
| @supabase/supabase-js | Supabase client |

## Media

| Package | Purpose |
|---------|---------|
| expo-av | Audio/video recording |

## Fonts

| Package | Purpose |
|---------|---------|
| @expo-google-fonts/caveat | Handwritten font |
| @expo-google-fonts/permanent-marker | Marker font |

## Installation Commands

```bash
# Core
bunx expo install react-dom react-native-web @expo/metro-runtime

# Gestures & Animation
bunx expo install react-native-gesture-handler react-native-reanimated

# Styling
bun add nativewind tailwindcss

# State
bun add zustand

# Backend
bun add @supabase/supabase-js

# Media
bunx expo install expo-av

# Fonts
bunx expo install @expo-google-fonts/caveat @expo-google-fonts/permanent-marker
```
